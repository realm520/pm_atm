# Polymarket 天气策略实盘上线清单

## 1) 交易与执行（必须）
- [ ] 真实下单接口联通（鉴权、签名、限频）
- [ ] 订单状态机可追踪（NEW/PARTIAL/FILLED/CANCELED/REJECTED）
- [ ] 幂等下单（client order id 去重）
- [ ] 下单失败自动重试（带退避）
- [ ] 撤单/超时撤单策略可用

## 2) 风控（必须）
- [ ] 交易前检查（仓位上限、事件敞口、总敞口）
- [ ] 日亏损熔断生效（触发后停止开新仓）
- [ ] 连续亏损冷却生效
- [ ] 单市场亏损熔断生效
- [ ] 人工 Kill Switch（可一键停机）

## 3) 稳定性（必须）
- [ ] WS 断线自动重连验证通过
- [ ] 重连后订阅恢复验证通过
- [ ] 进程重启后可从状态恢复（或安全重启）
- [ ] 时钟同步（NTP）正常

## 4) 观测与告警（必须）
- [ ] 结构化日志完整（trade/summary/error）
- [ ] 每分钟心跳指标可见
- [ ] 告警接入 Telegram（见告警阈值模板）
- [ ] 日终对账流程完成（本地 vs 交易所）

## 5) 发布策略（建议）
- [ ] 先跑 conservative 配置 3-7 天
- [ ] 达标后再切 standard 配置
- [ ] 单日只提升一次仓位参数
- [ ] 每次提升不超过 30%

---

## 资金建议（起步）
- conservative：$300-$1,000
- standard：$1,000-$3,000

> 建议账户资金 >= `max_total_notional` 的 5-10 倍。
